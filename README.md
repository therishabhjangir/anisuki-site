# AniSuki - simple site (React + Vite + Tailwind)

Follow the local setup steps below.

## Local preview
1. Install Node.js LTS from https://nodejs.org (run installer).
2. Open PowerShell and run:
   cd E:\anisuki-site
   npm install
   npm run dev
3. Open http://localhost:5173

## Edit content
- site config: src/siteConfig.js
- gallery images: put files in public/ and update src/components/Gallery.jsx
- posts: edit src/data/posts.json
